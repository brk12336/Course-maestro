# 📸 Dossier Images — CuisineOrder

## Comment remplacer les photos

Toutes les photos sont chargées depuis Internet (Unsplash) par défaut.
Pour utiliser TES propres photos, remplace les fichiers dans ce dossier.

## Format recommandé
- **Taille** : 400×280 pixels minimum
- **Format** : JPG ou PNG
- **Ratio** : 4:3 (paysage)

## Activer les images locales

Quand tes photos sont dans ce dossier, édite `js/products.js` :
Remplace `imgUrl:` par `image:` dans chaque produit, ou simplement ajoute tes fichiers — le site les chargera automatiquement si le fichier existe.

---
💡 **Conseil** : Si une image est manquante, le site affichera automatiquement une photo de remplacement depuis Internet.
